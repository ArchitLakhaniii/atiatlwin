# FlashRequest Backend Service

FastAPI backend service for FlashRequest marketplace.

## Deployment on Render

### Build Command:
```
pip install -r requirements.txt
```

### Start Command:
```
uvicorn backend.app:app --host 0.0.0.0 --port $PORT
```

### Environment Variables Required:
- `MONGODB_URI`: Your MongoDB Atlas connection string
- `DB_NAME`: Database name (default: flashrequest)
- `GEMINI_SERVICE_URL`: URL of your deployed Gemini service
- `JWT_SECRET_KEY`: Secret key for JWT tokens (auto-generated by Render)
- `JWT_ALGORITHM`: Algorithm for JWT (default: HS256)

## Local Development

1. Install dependencies:
```bash
pip install -r requirements.txt
```

2. Create `.env` file with required variables (see `.env.example`)

3. Run the server:
```bash
uvicorn backend.app:app --reload --port 8000
```
